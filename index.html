Chirag's Blog
My blog application with CRUD features.
This blog was created using Python Django. Static assets and image uploads are stored on AWS S3.
It uses technologies such as html, css, js, Django templating engine, bootstrap, google fonts, crispy forms, rich text ckeditor, PostgreSQL for the database, gunicorn server and is deployed on Heroku.
Please note: Stored on a free Heroku server that goes to sleep with inactivity, so it may take a few seconds to load the first page.

Wikipedia Bot
A chat-like application that uses the Wikipedia API to return the first couple sentences from Wikipedia.
Enter any topic to get information from Wikipedia.
Written in the Python Django Framework and stored on a Google Cloud Compute VM Instance.

COVID-19 PreScreen
This application was created to prescreen individuals for COVID-19. Prescreening logic was adapted from Infermedica.
This app is provided for informational purposes only and is not a replacement for clinical diagnosis or advice from medical professionals.

 
Gredisoft Bot
A fun little web app that listens to a users voice and uses the text for a google search.
In the future these voice commands could be used for semantic web searches (say by query of DBPedia or NELL).
Works best on Google Chrome.

 
Gredisoft Wikipedia Bot
An integration of the Gredisoft Bot (Webkit Speech Recognition and Synthesis) with the Wikipedia Bot (Wikimedia API) described below.
Works best on Google Chrome. Push the button and speak to get a voice response from Wikipedia.
Written in the Django Framework and stored on a Heroku dyno app server.
Please note: Stored on a free Heroku server that goes to sleep with inactivity, so it may take a few seconds to load the first page.
