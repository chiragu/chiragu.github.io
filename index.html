Gredisoft Bot

A fun little web app that listens to a users voice and uses the text for a google search. In the future these voice commands could be used for semantic web searches (say by query of DBPedia or NELL).
Requires a browser that supports the Web Speech API, such as Google Chrome.

Gredisoft Wikipedia Bot

An integration of the Gredisoft Bot (Webkit Speech Recognition and Synthesis) with the Wikipedia Bot (Wikimedia API) described above.
Push the button and speak to get a voice response from Wikipedia. Written in the Django Framework and stored on a Heroku dyno app server.
Please note: Stored on a free Heroku server that goes to sleep with inactivity, so it may take a few seconds to load the first page. Requires a browser that supports the Web Speech API, such as Google Chrome.
