Gredisoft Wiki Bot
An integration of the Gredisoft Bot (Webkit Speech Recognition and Synthesis) with the Wiki Bot (Wikimedia API) described below.
Works best on Google Chrome. Push the button and speak to get a voice response from Wikipedia.
Written in the Django Framework and stored on a Heroku dyno app server.
Live Site

Chirag's Blog
A blog application with CRUD features and user accounts.
Written in the Django Framework and stored on a Heroku dyno app server.
Note: Uses SQLite database. Heroku's clears SQLite databases daily, so there is no persistent storage.
In the future, I might update it to use PostgreSQL.
Live Site

Wiki Bot
A chat-like application that uses the Wikipedia API to return the first couple sentences from Wikipedia. Enter any topic to get information from Wikipedia.
This bot can be plugged into the Gredisoft Bot (see below) for a voice-based architecture.
Written in the Django Framework and stored on a Google Cloud Compute VM Instance.
Live Site

COVID-19 PreScreen
A Prescreen Questionnare for the COVID-19 Infection.
This application was created to prescreen individuals for COVID-19. Prescreening logic was adapted from Infermedica.
This app is provided for informational purposes only and is not a replacement for clinical diagnosis or advice from medical professionals.
Github Repo, Live Site

Gredisoft Bot
A fun little web app that listens to a users voice and uses the text for a google search. In the future these voice commands could be used for semantic web searches (say by query of DBPedia or NELL).
Works best on Google Chrome.
Github Repo, Live Site

Loc News
Locaion based news (currently country) using NewsAPI.
API Key needs to be added to locnews.py
Written in the Flask Framework with Python.
Github Repo
